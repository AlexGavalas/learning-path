Tools I have installed and how I use them

- [mise](https://github.com/jdx/mise) to install various versions of the same tool
- [killport](https://github.com/jkfran/killport) to kill a process running on some port
- [jq](https://github.com/stedolan/jq) to query JSON

## Shell

- [Ghost](https://github.com/ghostty-org/ghostty) for my terminal app
- [starship](https://starship.rs) for a pretty prompt
- [atuin](https://github.com/atuinsh/atuin) for my shell history

Some modern alternatives to essential tools

- [bat](https://github.com/sharkdp/bat), better `cat`, supports syntax highlighting
- [lsd](https://github.com/lsd-rs/lsd), better `ls`
- [raycast](https://www.raycast.com), instead of spotlight
