### Build UI components in isolation

[Storybook](https://storybook.js.org) can be used to build UI components in isolation. It is a development environment for UI components. It allows you to browse a component library, view the different states of each component, and interactively develop and test components.

Some potentially useful addons are:

- [Chromatic](https://www.chromatic.com)
- [Lost Pixel](https://lost-pixel.com)

Some alternatives to Storybook are:

- [Ladle](https://www.ladle.dev)
- [Whyframe](https://whyframe.dev)
- [React Styleguidist](https://react-styleguidist.js.org)
